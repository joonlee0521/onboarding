Overview: This readme describes the set of R scripts and data files for replication of "Sorting or Steering: The Effects of Housing Discrimination on Neighborhood Choice" by Christensen and Timmins.  

Instructions: 

(1) Scripts in this folder were written for R-Version: 4.04.  
(2) Scripts are organized according to the name of each Table or Figure as it appears in the publication.

Input data files required each script are defined below:

Figure 2: Figure_2_Trial_Maps; Data Files: "recsprocessed_JPE.rds" , "adsprocessed_JPE.rds" , "HUDprocessed_JPE_census_042021.rds"
